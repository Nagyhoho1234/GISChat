# Contributing to GIS Chat

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork and create a feature branch
3. Build the project (see [README](README.md#building-from-source))
4. Make your changes and test them in ArcGIS Pro
5. Submit a pull request

## Reporting Bugs

Use the [bug report template](https://github.com/Nagyhoho1234/GISChat/issues/new?template=bug_report.md) and include:
- GIS Chat and ArcGIS Pro version
- AI provider and model
- Steps to reproduce
- Log output from `%APPDATA%/GISChat/logs/`

## Guidelines

- Keep pull requests focused on a single change
- Follow the existing code style (C# conventions, nullable enabled)
- Test with at least one AI provider before submitting
- Do not commit API keys or `settings.json`
